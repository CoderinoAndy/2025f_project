About

The Proton web application organizes emails into four views (All Emails, Response needed, Read only, Junk, Junk Confirmation), and lets you view the full contents and summaries of emails, with the ability to draft replies and send them, all locally. This web application was developed by Andy Yu for the International Baccalaureate program as an individual assessment.

How to use
  • All Emails: shows everything by recency, with type and priority (stars).
  • Response needed: emails that require a reply.
  • Read only: informational emails that do not require a reply.
  • Junk Confirmation: uncertain junk mail that needs user confirmation.
  • Click any email row to open the expanded email view.
